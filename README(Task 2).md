# Handwritten Digit Recognition (MNIST)

## Overview
This project builds an AI model to recognize handwritten digits (0–9) using the MNIST dataset. A Convolutional Neural Network (CNN) is implemented using TensorFlow/Keras to achieve high accuracy in image classification.

---

## Features
- Uses MNIST dataset (handwritten digits)
- Preprocesses and normalizes image data
- CNN-based deep learning model
- Model training and evaluation
- Displays predictions with actual labels

---

## Technologies Used
- Python
- TensorFlow (Keras)
- NumPy
- Matplotlib

---

## Dataset
MNIST dataset:
- 60,000 training images
- 10,000 testing images
- Image size: 28×28 pixels (grayscale)
